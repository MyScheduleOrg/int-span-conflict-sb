On submit, compare two input timespans and return true if there is a conflict between the two and false if there is none. For example, if timespan A is 10:30am-2:30pm, and timespan B is 1:30pm-5:30pm on the same day, this should return true, because the timespan A and timespan B overlap. If there is no overlap, return false. Use the return value to display feedback in the UI. Documentation for the datetimepicker can be found here.
